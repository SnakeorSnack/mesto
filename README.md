**Интро**

Здесь будет проект Место.
В данном проекте вы можете добавлять новые карточки с изображениями и их названием, удалять их, ставить лайки, и редактировать профиль пользователя.

## Использованные технологии:
  - Nested BEM
  - HTML
  - CSS
  - Адаптивная верстка
  - JS
